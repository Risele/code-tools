# C-tools

Collection of C lang tools

## Measurements

Code for float-data processing: averaging, statistics. 
Moving average, low-pass, measuring slope, RMS, Puasson stats, resetting buffers at fast value changes, etc.


## fast_math.h

Fast bit-magic calculations of basic math functions (1/x, sqrt(x), 2^x, 1/sqrt(x)...)

## uint_strings.h

Macro to initialize constant uint_t array[] that store strings with ... strings.
